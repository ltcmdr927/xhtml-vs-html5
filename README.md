# XHTML vs HTML5

いわゆる Self-closing Tag が XHTML と HTML5 でどう違うのかというのを解説するためのサンプルです。

# 動かし方

1. リポジトリをクローン
2. $ bundle install
3. $ ruby app.rb
4. ブラウザで "http://localhost:4567/html5" と "http://localhost:4567/xhtml" にそれぞれアクセスしてみると…
